盒子属性

改变一个元素的显示方式。所有HTML元素都可以通过更改display属性将其显示方式设置为 block / inline / inline-block

  • display: block

    block：一个区块。block的元素会独占一行，左右不允许有元素与之并排，排序为由上至下。
    可以设置width和height

  • display: inline

    inline: 内联元素，不会独占一行。inline元素允许左右与其他inline元素并排，宽度根据包含的内容而定。
    width和height对inline元素无效。

  • display: inline-block

    既可以设置宽度和高度，又可以和其他inline元素并排。

举例

block
inline
inline-block

常见block元素

  • div
  • p
  • h1~h6
  • ul,li

常见inline元素

  • span
  • a
  • strong

BoxSizing

适用于所有能够设置width和height属性的元素。

  • box-sizing: content-box（默认）

    元素大小不包括border

  • box-sizing: border-box

    元素大小包含border

举例

border-box width 240px

content-box width 240px
